# Screenshots

Place the application screenshots referenced by the main [README](../../README.md)
in this folder, using these exact file names:

| File name | Screen to capture |
|---|---|
| `main-window.png` | Main window with a project open |
| `configuration-dialog.png` | New / Edit Configuration dialog |
| `new-project.png` | New Project dialog |
| `test-case.png` | Add Test Case dialog |
| `submissions.png` | Submissions view |
| `results.png` | Results view (after a run) |
| `analytics.png` | Analytics view |
| `settings.png` | Settings view |

PNG is recommended. Once added, the images render automatically in the README.
